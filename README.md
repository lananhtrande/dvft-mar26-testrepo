## Dark Theme Toggle

A toggle button has been added to the header.

- Clicking the button switches between light and dark mode
- The theme is controlled using a `data-theme="dark"` attribute on the `<body>`
- Styles are defined in `styles.css` using `[data-theme="dark"]`
